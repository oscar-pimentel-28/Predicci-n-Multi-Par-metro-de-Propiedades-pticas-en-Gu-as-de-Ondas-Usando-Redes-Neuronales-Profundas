# Predicci-n-Multi-Par-metro-de-Propiedades-pticas-en-Gu-as-de-Ondas-Usando-Redes-Neuronales-Profundas

Este trabajo presenta el desarrollo e implementación de un modelo de red neuronal profunda de múltiples salidas para la predicción simultánea de parámetros ópticos en guías de ondas.
